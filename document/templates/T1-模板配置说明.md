# T1 模板配置说明文档

## 模板概述

**模板名称**：叫主风格（t1）

**风格特点**：棕色/白色/橙色字体配色，搭配多种动画效果，适合知识分享、个人 IP 类视频。

**适用场景**：

- 个人品牌宣传视频
- 知识付费课程片头
- 专家讲师介绍视频
- 企业宣传片

### 字幕样式配置

模板定义了 4 种字幕样式，系统会根据语义自动选择：

| 样式名     | 用途       | 字体     | 颜色         | 字号 |
| ---------- | ---------- | -------- | ------------ | ---- |
| `default`  | 默认样式   | 微软雅黑 | 白色 #FFFFFF | 90   |
| `emphasis` | 强调内容   | 三尺季宋 | 橙色 #E6782D | 90   |
| `tertiary` | 第三句     | 三尺季宋 | 橙色 + 黑边  | 110  |
| `keyword`  | 关键词高亮 | -        | 橙色 #E6782D | -    |

| 屏数     | 特效      | 说明                   |
| -------- | --------- | ---------------------- |

| 第 2 屏  | snapZoom  | 快速放大               |
| 第 3 屏  | zoomOut   | 快速缩小               |
| 第 6 屏  | spotlight | 聚光灯效果（四周变暗） |
| 第 10 组 | zoomIn    | 渐进放大               |

### 标题配置

| 参数       | 值          | 说明            |
| ---------- | ----------- | --------------- |
| 位置       | top: 140    | 距顶部 140 像素 |
| 字号       | 110         | 字体大小        |
| 字体       | SanjiJiSong | 三尺季宋        |
| 颜色       | #EB8436     | 棕色            |
| 第二行颜色 | #FFFFFF     | 白色            |
| 描边       | 12px 黑色   | 文字描边        |
| 显示时间   | 0-6 秒      | 6 秒后自动消失  |

### 主讲人配置

| 参数     | 值                   | 说明        |
| -------- | -------------------- | ----------- |
| 名字位置 | left: 180, top: 1000 | 左下角区域  |
| 名字字号 | 50                   | 较大        |
| 名字颜色 | #E6782D              | 橙色 + 黑边 |
| 头衔字号 | 30                   | 较小        |
| 头衔颜色 | #FFFFFF              | 白色 + 阴影 |

### 画中画配置

动画效果

- `zoomIn` - 放大
- `zoomOut` - 缩小
- `panLeft` - 向左平移
- `panRight` - 向右平移
